# Commands
- /say [text]
- /ban [user]
- /unban [user_id]
- /purge [number]
- /mute [user/user_id]
- /unmute [user/user_id]
- /membercount 
# REQUIREMENTS:
- disnake 
```
pip install disnake
```
- discord-py-slash-command 
```
pip install discord-py-slash-command
```
